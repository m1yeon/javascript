17_객체(object)_내장객체_윈도우객체(브라우저객체)

• 브라우저 객체는 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체

• window객체는 브라우저 객체 중 최상위 객체로서 현재 실행되고 있는 브라우저창을 말함


1. window 객체의 속성

• document

• frameElement

• innerHeight

• innderWidth

• location

• name

• screenX

• screenY

• screenX, screenY

• sessionStorge

2. window 객체의 메서드

• alert() : 윈도우창에서 경고창(메세지)를 띄울 때 사용

• blur() : 현재창에서 포커스를 제거

• close() : 현재창을 닫기

• confirm() : 확인, 취소 버튼이 있는 확인창을 표시함

• focus() : 현재창에 포커스(초점)을 부여

• moveBy() : 현재창을 지정한 크기만큼 이동

• moveTo() : 현재창을 지정한 좌표로 이동

• open() :: 새로운 창을 열기

• prompt() : 사용자가 입력 가능한 프롬프트 창 열기

• resizeTo() : 동적으로 브라우저 창의 크기를 조절

• scroll() : 문서의 특정 위치로 스크롤 하기

• sizeToContent() : 내용에 맞게 창의 크기를 조절

• stop() : 로딩 중지
