01_자바스크립트 소개와 기초문법

자바스크립트는 html 문서안에서 동작되는 언어이다.

자바스크립트는 모든 명령어가 객체로 이루어진 언어이다.
HTML_태그, 속성 / CSS_선택자, 속성 / JS_객체, 속성, 매서드

html 문서의 head 또는 body 영역에 작성하며, script로 시작하여 /script로 닫아준다.

주석은 한 줄 주석 : / / 두 줄 주석 : /* */

html, css는 대소문자를 구분 하지 않지만, 자바스크립트는 대소문자를 구분한다.

자바스크립트에서 문자데이터를 입력하고자 할 때는 '', "" 기호를 사용한다.

자바스크립트는 css언어와 같이 내부스크립트, 외부스크립트가 있다.

자바스크립트의 문서확장자는 '파일명.js'이다.
